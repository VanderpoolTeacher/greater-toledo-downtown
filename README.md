# Greater Toledo Downtown

Web home of *Greater Toledo Downtown* magazine.

## Live site

Published via GitHub Pages: **https://vanderpoolteacher.github.io/greater-toledo-downtown/**

## Contents

- **`index.html`** — *A Decade Under Fire*, a data-driven feature on ten years of gun
  violence in Toledo, Ohio. Self-contained HTML; charts render via
  [Chart.js](https://www.chartjs.org/) and the hot-spot map via
  [Leaflet](https://leafletjs.com/), both loaded from a CDN. No build step.

More features will be added over time.

## A note on production

Portions of the feature were produced with AI assistance and reviewed by editors;
sourcing and methodology are described at the foot of the article.
